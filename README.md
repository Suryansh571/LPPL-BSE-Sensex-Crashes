# LPPL-BSE-Sensex-Crashes

**Reproducibility repository** for the paper:

**"Nonlinear Modelling of Financial Crashes: An Empirical Examination of Log-Periodic Power Law Signatures in Indian Equity Markets"**

### Authors
Tithi Mishra, Rajesh Mahadeva, **Suryansh Sunil**, Amit Kumar Goyal*, and Varun Sarda

### Contents
- `lppl_fit_multi_start.py` — Main Python script with multi-start optimization (40 runs), sensitivity analysis, and diagnostics
- `lppl_analysis.ipynb` — Jupyter Notebook version (recommended for interactive use)
- Five CSV files containing historical BSE Sensex data windows for the five major crashes (1990–2020)

### How to Run

**Using Jupyter Notebook (Recommended)**
